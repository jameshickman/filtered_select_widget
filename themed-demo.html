Filtered Select - Themed Examples

Theme Overview

The filtered-select component can be themed in three ways:

  1. Using the theme attribute with a predefined theme name
  2. Using the setTheme() method to apply a theme programmatically
  3. Creating a custom theme by extending the default theme

Default Theme

<filtered-select id="default-select"> <select> <option value="1">One</option> <option value="2">Two</option> <option value="3">Three</option> <option value="4">Four</option> <option value="5">Five</option> </select> </filtered-select>
Selected: None

Custom Theme (Created Programmatically)

<filtered-select id="custom-select"> <select> <option value="1">One</option> <option value="2">Two</option> <option value="3">Three</option> <option value="4">Four</option> <option value="5">Five</option> </select> </filtered-select> <script> // Create custom theme const pastelTheme = createFilteredSelectTheme({ backgroundColor: '#f9f7f0', textColor: '#5a5a5a', borderColor: '#e8e0d0', hoverBorderColor: '#c9af98', selectedBgColor: '#c9af98', selectedTextColor: '#fff', inputBorderColor: '#e8e0d0', borderRadius: '6px', fontSize: '15px', dropdownShadow: '0 4px 12px rgba(0,0,0,0.08)' }); // Apply the theme to the select element document.getElementById('custom-select').setTheme(pastelTheme); </script>
Selected: None

Dynamically Switch Themes

<filtered-select id="switchable-select"> <select> <option value="1">One</option> <option value="2">Two</option> <option value="3">Three</option> <option value="4">Four</option> <option value="5">Five</option> </select> </filtered-select> <script> function applyTheme(elementId, themeName) { const selectElement = document.getElementById(elementId); switch(themeName) { case 'default': selectElement.setTheme(FilteredSelectDefaultTheme); break; case 'dark': selectElement.setTheme(FilteredSelectDarkTheme); break; case 'rounded': selectElement.setTheme(FilteredSelectRoundedTheme); break; case 'custom': // Create a custom orange theme const orangeTheme = createFilteredSelectTheme({ backgroundColor: '#fff9f5', textColor: '#333', borderColor: '#ff9800', hoverBorderColor: '#e67e00', selectedBgColor: '#ff9800', borderRadius: '4px', dropdownShadow: '0 4px 12px rgba(255,152,0,0.2)' }); selectElement.setTheme(orangeTheme); break; } } </script>
Selected: None
